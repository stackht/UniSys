# UniSys

UniSys is a front-end prototype for an education management/ERP experience, including a landing page and an EduSys submodule.

## Structure
- `index.html` / `styles.css` / `index.js`: main landing page.
- `edusys/`: EduSys module pages and scripts.
- `edusys-preview.png`: landing page preview image.
- `ADMIN_EXTENSIONS.md`: admin extension documentation.
- `deep_clean_emojis.py`: utility script.

## Run locally
Open `index.html` in a browser.

## Notes
This repository currently contains static assets only; no build step is required.
