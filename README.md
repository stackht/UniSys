# UniSys

UniSys is a front-end prototype for an education management/ERP experience, including a landing page and an EduSys submodule.

## Structure
- `public/`: static site root (HTML/CSS/JS, `edusys/`, and `edusys-preview.png`).
- `docs/`: documentation (includes `ADMIN_EXTENSIONS.md`).
- `scripts/`: utility scripts (includes `deep_clean_emojis.py`).

## Run locally
Open `public/index.html` in a browser.

## Notes
This repository currently contains static assets only; no build step is required.
